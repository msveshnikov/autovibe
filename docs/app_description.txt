4.  **Witness Creation:** See the results unfold in real-time! AutoVibe displays the generated `README.md` (capturing the evolving idea) and a live `index.html` preview side-by-side in iframes.
5.  **Iterate & Evolve:** Behind the scenes, AutoVibe leverages the AutoCode CLI (autocode.work) to intelligently generate and refine code or content within each loop cycle, updating the preview files.
6.  **Stay in Control:** Monitor the iteration count and easily stop the process anytime with the prominent "Stop" button.
