How AutoVibe Works

1

Provide Seed & Key

Input your initial idea (seed), your Google AI API key, and select a model. The seed guides the process, the key enables it.

2

Kickoff & Iterate

Start the loop. The system creates a project space and iteratively runs AutoCode with the chosen model, refining content based on previous steps.

3

Preview & Stop

Watch the live previews update with each iteration. Stop the process anytime to review the current state.
